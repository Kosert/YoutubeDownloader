Frequently Asked Questions

Why do I get "Video is unavailable" error?

Make sure the video URL is correct and does not fall in any of the following:
  • Regionally restricted
  • Private
  • Rental
Since Youtube updates their website all the time, it is possible this will stop working at some point. In that case I will try to fix this as soon as possible.

What is the difference between MP4/WEBM/MKV?

  • MP4 is the MPEG-4 file format. MPEG-4 officialy does not support VP8/VP9 video formats and Vorbis audio format.
    If there is no VP8, VP9, Opus or Vorbis selected as either format, this is going to be the output container.
  • WebM is an open, royalty-free, media file format designed for the web. It supports only VP8/VP9 as video format and Opus/Vorbis as audio format.
    If you choose VP8/VP9 as the video format and Opus/Vorbis as audio format, this is going to be the output container.
  • MKV is the Matroska video file format. Matroska can contain audio and video tracks of any format that YouTube is using.
    If you choose any combination that is not compatibile with MP4 nor WEBM, this is going to be the output container.

Why the downloaded file has no extension?

Webm tracks hosted by Youtube have no file extensions added to file name. Since these files are hosted externally, I am not able to fix that.
You can add ".webm" extension to file manually.
